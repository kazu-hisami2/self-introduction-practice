# self-introduction-practice

## 概要

COACHTECH 教材 Tutorial 5-1「HTMLの基礎 ハンズオン演習」で作成した成果物です。
自己紹介とお問い合わせフォームを含んだWebページを作成しました。

## 使用技術

- HTML5

## 学んだこと

- フォームの基本構造やlabelのアクセシビリティ・属性の関連付け
- ブロックレベル要素とインライン要素の違いやレイアウトについて
- 画像の埋め込みや代替テキスト（alt="..."）の重要性

## 工夫したところ
- フォーム部分はブロックレベル要素で背景色を変化させわかりやすくしました。
- フォーム部分の入力欄のrequired部分を"(\*：必須)"とつけることでわかりやすくしました。

## 動作確認

- VScode内で開いたself-induction.htmlを右クリック⇒"Open in Itegrated Browser"をクリック。
- "self-introduction.html"ファイルをChromeにドラッグ。
