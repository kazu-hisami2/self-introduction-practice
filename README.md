# self-introduction-practice

## 概要

COACHTECH 教材 Tutorial 5-1「HTMLの基礎 ハンズオン演習」で作成した成果物です。
自己紹介とお問い合わせフォームを含んだWebページを作成しました。

## 使用技術

- HTML5

## 学んだこと

- フォームの基本構造や入力欄とlabelの関連付けについて理解できた。
- フォントの強調や背景色など、既存のフォーマットとは変化させてみました。
- 入力欄のrequired部分を"(\*：必須)"とつけることでわかりやすくしました。

## 動作確認

- VScode内で開いたself-induction.htmlを右クリック
  ⇒"Open in Itegrated Browser"で動作確認できました。
- "self-introduction.html"ファイルをChromeにドラッグした場合でも動作確認できました！
